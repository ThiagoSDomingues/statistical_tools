# statistical_tools
Tutorials to learn statistical tools that will be important to me during my life

1. ***PCA Tutorial:*** This comprehensive tutorial covers PCA from mathematical foundations to PhD-level applications, including manual implementation, scikit-learn pipelines, visualizations, and advanced variants.

* **Jupyter Notebook -- PCA_Tutorial.ipynb:** this notebook includes executable code blocks with Iris, Wine, and Digits datasets, scree plots, loadings analysis, and cross-validation for component selection. All cells run in standard Jupyter environments with scikit-learn, numpy, pandas, and matplotlib.
